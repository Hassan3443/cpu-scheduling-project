# CPU Scheduling Comparison Simulator

A modular, clean, and interactive simulator comparing **Round Robin** and **Priority** scheduling algorithms.

## Features
- Dynamic metric calculations (Turnaround Time, Waiting Time, Response Time).
- Interactive visualization using Gantt charts.
- Starvation detection systems for Priority scheduling.
- Deep-dive test scenarios with dynamic execution loading.
- 100% Vanilla JavaScript ES6 Modular setup.

## How to Run
This project relies purely on standard ES6 Module structures. No build steps (like Webpack or Node.js) are required.

1. Clone or download this project folder.
2. Open the project in **VS Code**.
3. Install the extension **Live Server** (by Ritwick Dey).
4. Right-click on `index.html` and select **"Open with Live Server"**.

*(Note: Because the application uses native `type="module"` JavaScript, it must be run over `http://` or `https://` via a local server, rather than the `file://` protocol.)*
